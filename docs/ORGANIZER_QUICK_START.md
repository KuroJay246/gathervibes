# Organizer Quick Start

## Start here

1. Sign in with the approved organizer account.
2. Open `Events` and either select the current Working Event or choose `Plan a New Event`.
3. Go to `Overview` to confirm the selected event, key numbers, and next actions.

## Working Event

The Working Event is the event currently shown across `Overview`, `Guests & Registrations`, `Payments`, `Tickets`, `Check-In`, `Operations`, `Message Builder`, and `Reports`.

If the wrong event is selected, change it in `Events` or use the event switcher in `Overview`.

## Your main pages

- `Overview`: event summary, finance snapshot, readiness, and quick actions.
- `Events`: create a new event, edit event details, manage tasks, and review readiness.
- `Guests & Registrations`: add or update registrations and guest counts.
- `Payments`: review charges, payments received, balances, and payment follow-up.
- `Tickets`: assign ticket codes and prepare QR-ready tickets.
- `Check-In`: search guests and record attendance on event day.
- `Operations`: track event-level expenses, commitments, suppliers, bakers, vendors, and sponsors.
- `Message Builder`: create and copy organizer messages. The app does not send them automatically.
- `Reports`: review read-only event follow-up, payment totals, Operations totals, and event summary.
- `Settings`: check workspace defaults and organizer settings.
- `System QA`: use safe QA guidance and release evidence without changing records.

## First planning checklist

1. Create the event in `Events`.
2. Confirm date, time, venue, and capacity.
3. Set the ticket price or price tiers.
4. Add registrations and record payments.
5. Add bakers, vendors, suppliers, sponsors, or helpers.
6. Track tasks and deadlines.
7. Prepare tickets and `Check-In`.
8. Review readiness in `Overview` and `Reports`.
